# fprint
Formatted printing for Python

## Usage

```
from fprint import fprint 

fprint(<"text">, <"text color">, <"text style">, <"background color">)
```

### Colors

- black
- red
- green
- yellow
- blue
- purple
- cyan
- white

### Styles

- bold
- underline
- italic
- blink
